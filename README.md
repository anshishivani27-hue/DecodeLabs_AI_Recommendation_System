# 🤖 AI Recommendation System

## 📌 Description

This project is a Python-based AI Recommendation System that suggests learning courses based on the user's interests, skill level, and daily study time. It uses rule-based recommendation logic to provide personalized suggestions and study plans.

## ✨ Features

- Interactive console interface
- Six technology domains
- Beginner, Intermediate and Advanced levels
- Personalized study plan
- Recommended learning platforms
- Continuous recommendations until user exits
- Input validation

## 🛠 Technologies Used

- Python 3

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python recommendation.py
```
